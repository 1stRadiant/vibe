Help Guide

How to Use Web Code Explorer

  • Upload: Drag-and-drop a .zip or .html file into the sidebar to start.
  • Edit Mode: Toggle 'Edit Mode' on to visually edit the site in the preview. Click elements to select them.
  • Element Tools: Once an element is selected, use the sidebar to change its style, content, and attributes.
  • Component Library: Drag components from the library and drop them into the preview window.
  • AI Tools: Use the AI chat for help, generate a new site from a prompt, or make site-wide edits automatically.
  • Save/Load: Use the "Save Project" and "Load Project" buttons to manage your work in progress. This functionality is handled by `app.js`.
  • Download: Click 'Review & Download' to inspect the final code in a professional editor and download your project.
